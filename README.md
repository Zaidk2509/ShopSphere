# ShopSphere 🛒

An E-Commerce Order and Inventory Management System.

## 🚀 Features
- Product & Inventory Management
- Order Processing System
- Payment Handling
- Stock Auto-Update
- Analytics Dashboard

## 🛠 Tech Stack
- Frontend: HTML / React
- Backend: Flask / Node.js
- Database: MySQL

## 📂 Project Structure
frontend/ → UI  
backend/ → API & Database  

## ▶️ How to Run

### Backend
cd backend  
pip install -r requirements.txt  
python app.py  

### Frontend
cd frontend  
npm install  
npm start
